## Getting starting

### Installation

Clone the repository:

```sh
git clone https://github.com/jeyofdev/angular-form-inputs
```

Change to the project directory:

```sh
cd angular-form-inputs
```

Install the dependencies:

```sh
npm install
npm i
```

Run Dev server

```sh
npm run start
```
